# File Browser

[File Browser](https://filebrowser.org/) provides a file managing interface within a specified directory.

!!! warning "For advanced users only"
    This package is useful to change files/settings without SSH.

## Permissions

To get write permission, the files/packages need to be in the **SynoCommunity** group, and be given +rw permissions for the group.

## Security Warning

!!! danger "DO NOT forward the port to the public internet!"
    Unless you've configured strong authentication, but even then it's not recommended.
