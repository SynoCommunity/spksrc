Subliminal

Subliminal allows you automatically download best-matching subtitles for your movies and tv shows on your DiskStation. This package is named after Subliminal, the Python library used to search and download subtitles.

Parameters

In this panel, you can edit your preferences and enable the automatic download.

General

  • Languages: Subtitles languages, in prefered order
  • Services: Services from which to download subtitles
  • Multiple subtitles: Enable the download of one subtitles file per language (.en.srt, .fr.srt)
  • Maximum depth: Maximum depth of sub-directories to scan in each configured directory
  • DSM Notifications: Enable DSM notifications after subtitles download

How To

  • If you select at least 2 languages, and do not check the multiple subtitles, it will try to download the first language, and if it doesn't exist, it will try to download the second language, and so forth. Using this method will name the file exactly the same as the video name
  • If you select at least 2 languages, and check the multiple subtitles, it will try to download every language you have set.

Task

Subliminal allows you to automatically download missing subtitles in custom directories on a daily basis.

  • Enable: Enable the daily automatic download of subtitles
  • Age: Videos older than this value will be ignored
  • Hour: Hour of the daily task
  • Minute: Minute of the daily task

Directories

This panel is used to customize directories containing videos you want to download subtitles for. You can set a parent directory regarding the maximum depth you set in the panel Parameters.

  • Name: A name for the Directory that you can use to find it easily in the list
  • Path: Full path to the directory, for example /volume1/video/TVShows

The button scan will run a scan of the selected folder(s) and download missing subtitles.
