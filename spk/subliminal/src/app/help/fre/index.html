Subliminal

Subliminal vous permet de télécharger automatiquement les meilleurs sous-titres pour vos films et séries sur votre DiskStation. Ce paquet est nommé d'après Subliminal, la librairie Python utilisée pour rechercher et télécharger les sous-titres.

Paramètres

Dans ce panneau, vous pouvez modifier vos préférences et activer le traitement automatique.

Général

  • Langues : Vous permet de choisir la (les) langue(s) des sous-titres, par ordre de préférence
  • Services : Services avec lesquels télécharger des sous-titres
  • Multiples sous-titres : Active le téléchargement d'un fichier de sous-titres par langue (.fr.srt, .en.srt)
  • Profondeur de scan : Profondeur maximale de sous-répertoires à scanner de chaque répertoire configuré
  • Notifications DSM : Activer les notifications DSM après le téléchargement de sous-titres

Tâche

Subliminal vous permet de télécharger automatiquement et quotidiennement les sous-titres manquants des répertoires configurés.

  • Activer: Activer ou non la recherche quotidienne automatique de sous-titres
  • Ancienneté: Les vidéos plus vieilles que le nombre de jours indiqués ne seront pas prises en compte
  • Heure: Heure de la tâche quotidienne
  • Minute: Minute de la tâche quotidienne

Répertoires

Ce panneau est utilisé pour indiquer les répertoires contenant les vidéos pour lesquelles vous désirez des sous-titres. Vous pouvez indiquer un répertoire parent selon votre règlage de profondeur de scan dans le panneau Paramètres.

  • Nom: Nom du répertoire, pour le retrouver facilement dans la liste
  • Chemin: Chemin complet du répertoire, par exemple : /volume1/video/TVShows

Le bouton Rechercher vous permet de lancer la recherche de sous-titres manquants dans le(s) répertoire(s) selectionné(s).
