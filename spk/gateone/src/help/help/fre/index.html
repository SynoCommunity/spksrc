GateOne

Description

GateOne est un client SSH en HTML5. Il est capable de créer plusieurs connexions simultanées sur un ou plusieurs serveurs.

Site web de l'éditeur : liftoffsoftware.com

Par défaut, GateOne écoute sur le port TCP 8022. Pour y accéder depuis internet, vouc avez deux possibilités : soit en router le port sur le routeur, soit en utilisant le paquet HAProxy qui permettra alors de disposer de GateOne derrière un même port 443 pour plusieurs services HTTPS (DSM, etc).
