Debian Chroot

Debian est un système d'exploitation (SE) qui rend disponible plus de 29000 paquets, logiciels précompilés et empaquetés dans un joli format pour rendre son installation facile sur votre DiskStation. Debian Chroot vous permet de bénéficier du SE Debian au sein de votre DiskStation, aux cotés de DSM. Ce paquet est destiné aux utilisateurs avancés uniquement.

Installation

Une fois l'installation terminée via le Gestionnaire de Paquets, elle continue en arrière plan et vous pourrez voir le statut Installation en courssous Vue d'ensemble dans le panneau de gauche. Lorsque l'installation en arrière plan est terminée, le statut passera à Installé.

Dans ce même panneau, vous pouvez aussi voir combien de services sont démarrés et effectuer les opérations de mise à jour.

Utilisation

Aussitôt que le statut est à Installé, vous pouvez commencer à utiliser le chroot. Pour ce faire, connectez vous en SSH (utilisateur root) à votre DiskStation puis tapez la commande suivante: /var/packages/debian-chroot/scripts/start-stop-status chroot.

Lors de la première utilisation, il est conseillé d'effectuer quelques opérations de configuration:

  • Mise à jour: à l'aide de apt-get updatepuis apt-get upgrade
  • Locales: avec apt-get install localespuis dpkg-reconfigure locales
  • Fuseau horaire: en exécutant dpkg-reconfigure tzdata

Configurer les services

Debian Chroot vous permet de gérer les paquets que vous avez installé directement depuis DSM.
