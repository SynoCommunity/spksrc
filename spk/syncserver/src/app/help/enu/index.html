Mozilla Sync Server

You can use Mozilla Sync Server to synchronize your bookmarks, passwords, settings, history, add-ons and tabs with Firefox on other computers. Set up Firefox Sync on your main computer by creating a new account on a custom server, then add your other computers.

Which URL should I use to configure Firefox Sync on my computer?

The IP of your Synology, along with port 8084: http://syno-ip:8084/. Alternatively, if you have name resolution configured correctly, you can use http://my.domain.name:8084/

How do I reset my password?

Open your browser, and enter http://syno-ip:8084/weave-password-reset.

This opens a web page where you can enter your e-mail address. A link to reset your password will be sent via e-mail.

How do I remove an account?

Open your browser, and enter http://syno-ip:8084/weave-delete-account.

This opens a web page where you can enter your e-mail address and password to delete an existing account.

FAQ

Why do I see "404 Not Found - The resource could not be found" when I open the url directly in my browser?

This is expected behaviour. The webserver used for this service does not serve pages on http(s) connections, so a 404 is displayed.

I want to change the IP of my Synology. What do I do?

Adjust the fallback_node in syncserver.conf with the correct IP-address.

I am using the built-in SSL, and I want to change the certificate

First, create a .pem file. You can do this by running the command "cat server.crt server.key > server.pem". Note that a line break between the two is needed.

Replace the server.pem in /usr/local/syncserver/var/. Make sure the owner is set correctly: "chown -R syncserver:root server.pem", and set the access permissions to owner/read: "chmod 400 server.pem"
