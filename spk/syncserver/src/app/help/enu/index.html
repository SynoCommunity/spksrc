Firefox Sync Server 1.1

You can use Firefox Sync Server to synchronize your bookmarks, passwords, settings, history, add-ons and tabs with Firefox on other computers. Set up Firefox Sync on your main computer by creating a new account on a custom server, then add your other computers.

Firefox Sync Server 1.1 is compatible with FF30 and older, and can also be used on mobile clients and forks.

Which URL should I use to configure Firefox Sync on my computer?

The IP of your Synology, along with port 8084: http://syno-ip:8084/. Note the slash at the end.

Alternatively, if you have name resolution configured correctly, you can use http://my.domain.name:8084/.

Note: The webserver used for this service does not serve pages on http(s) connections, so a 404 is displayed when you type the URL into your browser.

How do I reset my password?

Open http://syno-ip:8084/weave-password-reset in your browser.

This opens a web page where you can enter your e-mail address. A link to reset your password will be sent via e-mail.

How do I remove an account?

Open http://syno-ip:8084/weave-delete-account in your browser.

This opens a web page where you can enter your e-mail address and password to delete an existing account.

I changed the IP of my Synology. Now what?

Reinstall the package, or alternatively, adjust the fallback_node in "/usr/local/ffsync/var/ffsync.conf" with the correct IP-address.

I am using the built-in SSL option, and I want to update the certificate

First, create a .pem file. You can do this by running the command "cat server.crt server.key > server.pem". Note that a line break between the two is needed.

Replace the existing server.pem file in "/usr/local/ffsync/var/". Make sure the owner is set correctly: "chown -R syncserver:root server.pem", and set the access permissions to owner/read: "chmod 400 server.pem"
